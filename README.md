# Ứng dụng EPL chat
Sử dụng công nghệ HTML, CSS và Nodejs (Thư viện Socket.io).

## Các bước chạy hệ thống

### Đầu tiên chạy lệnh 
### `npm install`

### Sau đó chạy lệnh
### `npm start`

Chạy ứng dụng ở chế độ development.\
Mở [http://localhost:8080](http://localhost:8080) để xem hệ thống trên trình duyệt.
